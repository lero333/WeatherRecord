## Описание
Training Planner - это графическое приложение для планирования и отслеживания тренировок. Приложение позволяет:

- Добавлять тренировки с указанием даты, типа и длительности
- Просматривать список всех тренировок в таблице
- Фильтровать тренировки по типу и дате
- Удалять тренировки
- Автоматически сохранять и загружать данные в JSON формате
- Проверять корректность вводимых данных

## Требования
- Python 3.7+
- Стандартные библиотеки Python (tkinter, json, datetime)
